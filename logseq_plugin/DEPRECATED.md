# Logseq Plugin Deprecated

This entire directory is deprecated as of Cymbiont 1.0.

The Logseq plugin provided:
- Real-time bidirectional sync
- WebSocket communication 
- Graph switching coordination
- Config.edn manipulation

All of this complexity is being removed in favor of:
- Import-only PKM support
- Terminal-first agent interface
- Clean Unix pipe architecture

This directory will be removed in the next major version.
